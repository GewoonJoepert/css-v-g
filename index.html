Over mij

Mijn naam is Joep Vernooij en ik ben 14 jaar oud. Ik zit op het Metis Montessori Lyceum (MML). Op het Metis is er een extra keuzevak genaamd de coderclass. Coderclass is een profiel waarbij je leert programmeren in veel verschillende computertalen zoals: Python, HTML-CSS & javascript. Deze website heb ik gemaakt met HTML-CSS. Verder heb ik ook hobby's zoals

Badges

In de coderclass heb je een beloningssysteem: badges. Badges zijn beloningen die je krijgt als je een programmeertaal zoals python of javascript voldoende beheerst. Badges kan je verdienen als je alle opdrachten, de eindopdracht & de goedkeuring van een docent hebt. Aan het eind van het jaar moet je dan een aantal badges hebben zodat je genoeg weet voor het volgende jaar.
